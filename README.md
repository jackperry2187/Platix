# Platix
A 2D Unity Platformer  
Jackson Perry 2/6/2019

## Setup
Clone the repository, open the `Version 1.0` folder, and double click `Platix.exe`  
Controls and features can be found in `README.md` file in the `Version 1.0` folder. 

## Skills Used
- Unity
- C#
- Textures
- Texture Maps
- Unity Physics
